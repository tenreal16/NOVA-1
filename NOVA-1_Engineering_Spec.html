AI COMPUTE ARCHITECTURE // CORE INTELLIGENCE STACK
SOC // BRAIN
Qualcomm Snapdragon 8 Gen 3 or MediaTek Dimensity 9300 with dedicated NPU. Handles all on-device inference at hardware speed. No cloud required.
45 TOPS NPU 4nm NODE 12GB LPDDR5X
LLM // ON-DEVICE
Quantized 3B–7B parameter model (LLaMA 3.2 / Gemma 2 / Phi-3). Runs fully offline in INT4/INT8 quantized form on 256GB UFS 4.0 internal storage.
7B PARAMS INT4 QUANT 256GB UFS 4.0
VOICE // TTS+STT
On-device Speech-to-Text via Whisper Tiny/Base. TTS via Coqui or Kokoro with selectable voice modules — jarvis, female, custom. Max 5s response.
WHISPER BASE KOKORO TTS 5 VOICE MODES
JARVIS RESPONSE PIPELINE
01
WAKE WORD DETECTION
Always-on DSP listens for "Hey NOVA" at <5mW. Triggers main SoC wake. Beamforming mics isolate voice direction.
02
SPEECH → TEXT (STT)
Whisper Base processes audio locally in ~0.3s. No cloud upload. Works offline. Emotion tone is extracted in parallel.
03
EMOTION + CONTEXT ENGINE
Sentiment analysis classifies user state: stressed, excited, curious, tired. Adjusts LLM system prompt and TTS tone accordingly in real-time.
04
LLM INFERENCE + MEMORY
On-device LLM receives prompt + injected memory context. Generates response. NPU handles token inference at ~30 tok/s.
05
INTERNET AUGMENT (IF NEEDED)
WiFi/SIM used for live queries only. NOVA decides if internet is needed. If yes → fetch → inject context → re-infer. Adds ~1-2s max.
06
TTS OUTPUT → EARBUDS
Kokoro TTS renders voice in selected module. Auto-streams to slide-out earbuds via BT LE Audio. Sub-50ms latency. Total pipeline: <5 seconds.
ADAPTIVE MEMORY SYSTEM
SHORT-TERM MEMORY
Full conversation context in-session (last 8K tokens). Knows what you said 10 minutes ago. Resets on sleep.
LONG-TERM MEMORY
Vector database (ChromaDB) on-device. Stores user preferences, habits, relationships, schedule patterns, reaction history. Grows daily.
BEHAVIOURAL LEARNING
Tracks: preferred topics, emotional patterns, daily routine, response style. Fine-tunes system prompt per user. NOVA becomes you-specific over time.

ENGINEERING CHALLENGES // RISK MATRIX
CHALLENGE
ENGINEERING SOLUTION
DIFFICULTY
7B LLM in 2×2×6 IN box
INT4 quantization on Snapdragon NPU. LLaMA 3.2 3B fits in 2GB RAM. 7B needs INT4 + offload to UFS flash.
⬛⬛⬛⬛⬜ HARD
Thermal in compact chassis
Vapor chamber cooling + graphene thermal pad. Throttle inference during sustained load. Fan-less design possible at 3B model size.
⬛⬛⬛⬜⬜ MEDIUM
<5 second total pipeline
Parallel STT + context load. Pre-generate first TTS token while LLM still running. Streaming TTS output. Realistically achievable at 3B model.
⬛⬛⬛⬜⬜ MEDIUM
IP68 + SIM slot + USB-C
Silicone-gasketed SIM tray + rubber USB-C flap. Use magnetic pogo-pin charging as alternative. Audio jack routed through sealed membrane.
⬛⬛⬜⬜⬜ MEDIUM
Slide-out earbuds BT auto-pair
Hall sensor triggers BT pairing on slide. Earbuds pre-paired in factory. Magnetic rail slide mechanism. Charging via pogo pins in bay.
⬛⬛⬛⬜⬜ MEDIUM
On-device adaptive learning
User profile as system prompt injection (no retraining needed). LoRA fine-tune weekly on user data with on-device training. ChromaDB for memory vectors.
⬛⬛⬛⬛⬜ HARD
Battery life vs AI compute
DSP handles wake word at 5mW. NPU-only inference for voice. WiFi/SIM off by default. 4000mAh gives ~12h mixed use.
⬛⬛⬜⬜⬜ MEDIUM
